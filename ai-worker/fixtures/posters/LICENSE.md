# Anime poster fixtures — license note

These PNG files are synthetic, programmatically rendered placeholders
generated by `scripts.make_anime_poster_fixtures`. They contain no
copyrighted artwork, characters, or trademarks — only basic shapes,
gradients, and Korean text matching the `anime_kr.jsonl` doc IDs.

License: CC0 / public domain. They exist purely to give the multimodal
eval harness a non-empty image+text payload to exercise.
